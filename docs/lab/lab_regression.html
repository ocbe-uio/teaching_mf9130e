### Fit a logistic regression model for `low` and `age`

```{r}
#| label: birth example (2)
#| warning: false
#| echo: true

# low ~ age 

lr_low_age <- glm(low ~ age, data = birth_data, family = 'binomial')
summary(lr_low_age)

```


```{r}
#| label: birth example (3)

```
